## Lore del joc

El joc transcorre en una nit estranya on la realitat es desencaixa: són les 3:33 AM i el mòbil vibra una vegada i una altra. No és una notificació normal. El botó de bloqueig no respon i la protagonista sent que cada vegada que intenta aturar-ho… no ho decideix ella.

A mesura que avança, la pantalla s'estira, el soroll canvia i el temps es congela. El cos pesa, com si estigués caient “cap endins”. En aquest malson digital, les xarxes socials prenen forma física i la persegueixen.

L'única escletxa per avançar és reunir tres roses disperses pel nivell. Quan les recull totes, les roses bateguen alhora, el temps s'atura i la pantalla s'obre, fent-la caure a la planta següent.
